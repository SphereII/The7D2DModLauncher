Why a 7 Days To Die Launcher?

The 7D2D Launcher was created to help 7 Days To Die players and server administrators keep their moddified files updated.

One of the most frustrating aspects of connecting a modded 7 Days to Die server was getting out of date, or missing, icons or requiring extra mods to play. Players would have to seek out forum posts or server administrators in order to get information about the latest versions reqiored by the server. In popular forum threads, this information could span across several pages, and not always easy to find.

With no current Steam Workshop available, users must resort to a series of forum posts across various web sites, in various formats, and confusing instructions.

The 7D2D Launcher is aimed at creating a more uniform experience to players, admins, and modders.

What does the 7D2D Launcher do?

The 7D2D Launcher will load an XML file, that is maintained by your server administrators. This file will contain download links and other server information. The 7D2D Launcher will display the relevant information to you, allowing you to select a server to connect to.

Once you have selected an individual server, the 7D2D Launcher will make a copy of an existing copy of your 7 Days to Die, or will let you download a fresh copy from Steam. All you have to do is choose where you want to get the files from, and where you want them to go.

The 7D2D Launcher will keep a copy of the game seperated for each server you wish to connect too. This will allow you to quickly change servers, without worrying about changing or updating mods.

The 7D2D Launcher will download each of the mods fresh each time you play, ensuring that you are using the latest version, each time you play.

How can I use it?

You can find the link to the installer in the Download section. If you already have the 7D2D Launcher installed, it will check for updates every time you start it.

Also available is YouTube Video Tutorial: https://www.youtube.com/watch?v=PNpVsRDpCJQ&ab_channel=7D2DLauncher

If this is your first time using the 7D2D Launcher...

  • Get the link to your server's 7D2D Configuration file, usually by reviewing the group's forum posts
  • Set your Destination Folder for Mods, by clicking on the Blue folder
  • Click on File, then Open URL
  • Copy and paste the link to the 7D2D Configuration file
  • Choose which Server Group and Server you want to connect to.
  • Create a Copy from a locally installed copy of 7 Days to Die, or download from Steam
  • Press the Play button to get the latest versions of the Mods, and start playing

FAQs

Where does 7D2D Launcher see itself compared to Steam Workshops?

Steam Workshop opens up a huge market place for modders, allowing an easy to deploy and manage modding environment.

However, Steam Workshop is not yet supported by 7 Days to Die. The 7D2D Launcher aims to fill that gap by providing an easy to use and manage deployment strategy while we wait for official Steam Workshop support.

When Steam Workshop support does get released for 7 Days to Die, the 7D2D Launcher will still be available, and work with existing and new mods, for as long as users continue to use it.

Where can I download the latest version of the 7D2D Launcher?

The installed 7D2D Launcher will automatically check for updates when it starts.

The direct download link can be found here: https://7d2dlauncher.blob.core.windows.net/installer/7D2DLauncher.application

Will the 7D2D Launcher affect any of my current files or mods?

The 7D2D Launcher will create a copy, leaving the originals unchanged. This will allow you to have multiple configurations on your computer, allowing you to switch to them quickly.

Whoa! Just how much disk space is this going to cost?

The base 7 Days to Die game takes a little over 3 gigs of disk space. You'll need atleast 3.5G available in order to run each new copy.

Who are the developers of the 7D2D Launcher?

I'm a system analyst who felt that playing on modded servers was more of a hassle then necessary. I designed and developped a tool to help resolve some of the most routine tasks.

Is this an officially supported tool from The Fun Pimps?

This tool was developped independently from The Fun Pimps, and is unofficial.

Does this tool work with every mod?

The 7D2D Launcher was designed to deploy downloadable mod files, and install them in the correct play. If a modder structures their downloadable modded file in a standard format, the mod should play.

Does this tool help me make mods?

The 7D2D Launcher does not create any mods or change any mods. It simply uses pre-existing mods made and released independently by modders.
