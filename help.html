Quick Start Tutorials

How do I find 7D2D Launcher supported servers?

The 7D2D Launcher is meant to be decentralized, therefore cannot be used to discover new servers. However, searching through the various forums for groups that are advertising for 7D2D Launcher support.

For example, Forum Fanatics supports the 7D2D Launcher.

How can I find 7D2D Launcher supported mods?

The 7D2D Launcher is meant to be decentralized, therefore cannot be used to discover supported mods. However, search through the various forums in the Mods section for modders advertising the support.

Some mods will work without any modifications. The following are examples that we have tested, and are listed here without implying their endorsement of 7D2D Launcher:

  • Valmod Expansion Pack
  • A Clockwork Project

FAQs

The direct download link can be found here: Dwonloads

I'm still lost. How can I add a mod to My Mods?

We will use A Clockwork Project, by 7 Days to Die Forum's Clockwork Orange as an example.

  • Visit Clockwork Orange's forum link here
  • Scroll down until you see the download links:
      14.4 Stable: Download
      A 14.6 Stable and Experimental: Download
      UI Version Pack: Download Currently only Stable vanilla version available!
    In the Forum post, right click on the Download link by "A 14.6 Stable and Experimental: Download".
  • In the 7D2D Launcher, right click in the Server / Select Mod, and select "Add new My Mod"
  • Change "My Server" to "A Clockwork Project"
  • In My Server Description: A Clockwork Project Mod
  • Click on Add New Download Link, and paste in the URL from the Download link mentioned above.
