7D2D Launcher

The 7D2D Launcher provides an effortless way to play on your favorite modded servers:

  • Stay updated with the latest version of mods
  • Automatically set Server Address and Ports
  • Keep Mods seperated. Each server has its own isolated environment
  • For Admins, a central configuration for server administerators to update
  • For Players, a single file to view their favorite Server Group's servers

Using a central configuration file, server administrators can add all their individual servers, setting which mods are required for each one.

Players connect to that configuration file, and download the latest version of those mods, every time.
