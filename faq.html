Where can I download the latest version of the 7D2D Launcher?

The 7D2D Launcher will automatically check for updates when it starts.

The direct download link can be found here: https://7d2dlauncher.blob.core.windows.net/installer/publish.htm

Will the 7D2D Launcher affect any of my current files or mods?

The 7D2D Launcher will create a new folder, to generate each new environment. This will allow you to have multiple configurations on your computer, allowing you to switch to them quickly.

Whoa! Just how much disk space is this going to cost?

The base 7 Days to Die game takes a little over 3 gigs of disk space. You'll need atleast 3.5G available in order to run each new copy.

As a Server Administrator, this seems like a lot of extra work.

There is a small administrative overhead when maintaining a 7D2D server configuration. You will need to update it when you post a new zip file, or if any other information changed.

Who are the developers of the 7D2D Launcher?

I'm a system analyst who felt that playing on modded servers was more headache then necessary, so I developer a tool to help resolve some of the most routine tasks.

Is this an officially supported tool?

This tool was developped independently from The Fun Pimps, and is unofficial.

Does this tool work with every mod?

The 7D2D Launcher was designed to generate a seperate modded instance of 7 Days to Die, without impacting other installs. If a modder generates a valid Server Configuration file, then the mod should deploy.

Does this tool help me make mods?

The 7D2D Launcher does not create any mods, or changes any mods. It simply uses pre-existing mods made and released by modders.
